## Tools to Use
- Mac: Cmd+Shift+4 (drag to select 1270x760)
- Windows: Snipping Tool (set custom size)
- Browser: Use responsive design mode (1270x760)
- Upscale: Use Figma or Canva to resize if needed

## Editing Tips
1. Hide browser chrome (Cmd+Shift+F for fullscreen)
2. Use dark mode for consistency
3. Add subtle drop shadow in post
4. Compress with TinyPNG before upload
5. Ensure text is readable at thumbnail size
